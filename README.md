STEP 1: 기초 데이터 구조 설계


Product 클래스를 생성하여 상품명, 가격, 재고, 설명 필드 정의

ArrayList를 활용해 여러 개의 상품 객체를 하나의 바구니에 담기

for 반복문을 사용하여 저장된 상품 목록을 화면에 출력


STEP 2: 역할 분리


CommerceSystem 클래스를 도입

Main에서 데이터를 전달

start() 메서드를 구현하여 프로그램의 실행 흐름(입력/출력)을 독립시킴


STEP 3: 시스템 고도화 및 카테고리 관리


Category 클래스를 추가하여 상품을 그룹별(전자제품, 의류 등)로 관리

Customer 클래스를 통해 누적 구매 금액에 따른 자동 등급 산정 로직 구현

중첩 반복문을 활용한 '상세 메뉴 보기' 및 '뒤로 가기' 기능 구현
